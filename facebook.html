Facebook Touchtable

In this project we used a MultiTaction touch table and Unity to create an interactive data visualisation app for Facebook. Acrylic pucks with markers beneath them were used to trigger different info panels for sports. The table was exhibited at an event in France and then in the offices of Facebook. I worked as a freelancer and did the programming, while Daniel Lambert worked on the UX and the art. This was a project with an extremely fast turnaround.
